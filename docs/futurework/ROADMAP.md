# Roadmap

## Now — V1 submission-ready
- Simple native dictation
- On-device/offline-after-setup speech
- Light local cleanup
- Tiny onboarding
- Mac App Store proof

## Later — only after real users validate V1
- Tested additional language support
- Optional audio-file transcription
- Carefully scoped advanced controls
- iOS exploration only if the core customer demand proves it

Anything not in “Now” is intentionally not being built.
