## App Review notes template

```text
VachaVox is a paid macOS dictation utility with no account, no backend, and no
in-app purchases.

Review path:
1. Launch the app.
2. Complete the short onboarding.
3. Approve microphone access when prompted. This is used only for live local
   dictation.
4. The app prepares an offline speech capability for the Mac. Once ready, normal
   dictation works without a network connection.
5. Accessibility permission is requested only to paste the finished text into
   the app that was focused before dictation. If it is not granted or is
   unavailable, VachaVox copies the final text to the clipboard instead.

The app does not upload audio or transcripts. No test account is required.
```
